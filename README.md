# HelpTodo
서비스 URL : www.helptodo.site

## HelpTodo 서비스

캘린더를 통해 일정을 관리할 수 있는 투두리스트 웹 서비스입니다. 팀 단위로 일정을 공유하고, 팀원들과의 일정을 캘린더를 통해 한눈에 확인할 수 있으며, 변경사항들은 팀원들에게 실시간으로 반영됩니다.
